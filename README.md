
# My Trading System - Grid of Charts

This Next.js app displays 10 TradingView charts in a 5x2 grid layout, each labeled by timeframe.

## Timeframes Displayed:
- 3Min, 5Min, 15Min, 30Min, 1HR
- 2HR, 4HR, Daily, Weekly, Monthly

## How to deploy

1. Push to GitHub.
2. Connect to Vercel.
3. Deploy.
